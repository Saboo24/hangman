## 🎬 GAME OVERVIEW

A modern reimagining of the classic Hangman —
built with a **fast-paced timer system** and a **clean gaming interface**.

⚡ You don’t just guess…
⏱️ You race against time.

💀 One mistake closer to defeat
🟢 One correct guess keeps you alive

---

## 🎮 CORE FEATURES

* ⏱️ **10-second countdown system**
* 🎯 **6 progressive levels**
* 🧠 Smart word logic
* ⌨️ Interactive keyboard UI
* 🟢 Victory animations (neon green)
* 🔴 Defeat effects (shake + red aura)
* 🗺️ Level selection map
* 🎬 Smooth transitions & cinematic feel
* 🌑 Minimal dark UI (gaming aesthetic)

---

## 🧠 HOW TO PLAY

```text id="zx5d4k"
→ Guess the hidden word  
→ Each correct letter resets the timer  
→ Wrong guesses bring you closer to defeat  
→ Time runs out = penalty  
→ Survive until the end... if you can 💀
```
